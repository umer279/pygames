## 🛠 1. Environment Setup

To keep your system clean, we will use a **Virtual Environment (venv)**. Open your terminal or command prompt in the folder where your script is located.

### Create the Virtual Environment
- **Windows:** `python -m venv venv`
- **Mac/Linux:** `python3 -m venv venv`

### Activate the Virtual Environment
- **Windows:** `.\venv\Scripts\activate`
- **Mac/Linux:** `source venv/bin/activate`

### Install Dependencies
Once activated, install the mouse control library:
```bash
pip3 install pygame
```

## 🚀 2. Running the Games
```bash
python3 game_name.py
```
